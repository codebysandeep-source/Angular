Pipes


Pipes are basically used for transfer the data from one format to another format. 

For example - 

you have a string and you want to just make it in uppercase or in the lowercase that time you can use pipes,
or you want to update a date format or you want to apply the percentage on any number or you want to apply the local  in that all cases we can use the pipes




Using pipes: 


    
  
    String
    
  
  • normal -> {{myText}}
    • 
  
  • titlecase -> {{myText | titlecase}}
    • 
  
  • uppercase -> {{myText | uppercase}}
    • 
  
  • lowercase -> {{myText | lowercase}}
    • 





    
  
    Date
    
  
  • normal -> {{todayDate}}
    • 
  
  • date -> {{todayDate | date}}
    • 
  
  • date:'fulldate' -> {{todayDate | date:'fullDate'}}
    • 





    
  
    Slice
    syntax-> slice: from:to 
    example-> slice: 0:5    
  
  • normal text -> Hello Developers!
    • 
  
  • slice: 0:5 -> {{myText1 | slice: 0:5 }}
    • 
  
  • slice: 0:9 -> {{myText1 | slice: 0:9 }}
    • 
  
  • slice: 2:10 -> {{myText1 | slice: 2:10 }}
    • 





    
  
    JSON
    
  
  • normal -> {{myList}}
    • 
  
  • json -> {{myList | json}}
    • 





    
  
    number
    syntax-> number: 'leftNumberDigit . 1-noOfDigitAfterDot' 
    example-> number: '1.1-3'    
  
  • normal -> 007.43500
    • 
  
  • number:'1.1-3' -> {{007.43500 | number: '1.1-3'}}
    • 
  
  • number:'2.1-1' -> {{007.43500 | number: '2.1-1'}}
    • 
  
  • number:'5.1-2' -> {{007.43500 | number: '5.1-2'}}
    • 





    
  
    currency
    
  
  • normal -> 100
    • 
  
  • currency: 'USD' -> {{100 | currency:'USD'}}
    • 
  
  • currency: 'INR' -> {{100 | currency:'INR'}}
    •
